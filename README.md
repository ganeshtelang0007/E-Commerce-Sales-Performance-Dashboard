# E-Commerce-Sales-Performance-Dashboard
Project Overview

This project focuses on analyzing e-commerce sales data using Power BI. The dashboard provides insights into revenue, orders, customers, seller performance, and sales trends to support business decision-making.

Tools Used
Power BI
Power Query
DAX
Excel
KPIs
Total Revenue: 13.59M
Total Orders: 99K
Total Customers: 96K
Average Order Value: 136.68
Dashboard Features
Monthly Revenue Trend
Orders by State
Order Status Analysis
Top Customer Cities
Revenue by Seller State
Interactive Filters and Slicers
Key Insights
Total revenue reached 13.59M from nearly 99K orders.
São Paulo recorded the highest number of customer orders.
SP generated the highest revenue among seller states.
More than 97% of orders were successfully delivered.
Revenue showed steady growth throughout the observed period.
Customer demand was highly concentrated in major cities.
The average order value remained at 136.68 per order.
Revenue distribution suggests opportunities for expansion in lower-performing regions.
Conclusion

This dashboard helps businesses monitor sales performance, customer behavior, and regional trends, enabling data-driven decision-making.
